两种去向

私人积累和公开补题分开处理

  1. 只留给自己点保存后立即回到“我的题目”，可以编辑、搜索、练习和导出。
  2. 公开给大家先通过隐私检查并去 GitHub 查重；确认没有相同题目后，回到题目卡新建并在 GitHub 点击提交。改回“只留给自己”不会撤回已提交的内容。
  3. 其他人参与公开题目可以被看到和评论；正式题库里的每一道题下方也有自己的评论区。
