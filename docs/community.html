社区流程

帖子立即公开，正式题库单独收录

这里没有让投稿者困惑的“待审核页面”。社区帖子提交后就能看和评论；只有进入正式题库前，才会检查隐私、来源和技术事实。

  1. 01普通表单投稿不需要写 Markdown；不知道答案时只填问题即可。
  2. 02公开讨论补充每个投稿都是一条 GitHub 讨论线程，可继续评论和修正。
  3. 03整理进入题库确认内容可公开、事实可靠、表达适合面试后，再收录成正式题目。
先说明两件事

公开社区与私人笔记不能混在一起

公开仓库

任何人可看，登录 GitHub 后投稿和评论

帖子、用户名、附件和评论都会公开。请使用匿名背景，不要上传公司内部题库、面试官信息、联系方式或 NDA 内容。

私有仓库

只有获授权的仓库成员能参与

静态网页不会索要 Token，也无法匿名读取私有 Issue。私人面经继续留在浏览器本地或你自己的 Private 仓库。

“参考答案待校对”不是“题目待发布” 这类题目已经可以阅读和模拟，只是答案还没有逐题对照一手资料。你可以直接从题目页发起校对帖。
