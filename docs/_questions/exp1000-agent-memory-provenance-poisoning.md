## 面试时怎么答

要求每条记忆带来源、置信和确认状态，更新采用版本而不是覆盖原文。

**可以这样答：**

> 长期记忆不应只是自由文本，每条事实附原始消息或工具证据、主体、时间和是否经用户确认。新信息冲突时创建新版本并降低旧条目的有效性，不直接把历史改成看似一直正确。发现污染后可按来源批量撤销派生记忆，并重算受影响摘要和决策。高风险事实设置过期或每次使用前重新验证，不能无限期信任。

## 常见追问

1. **用户说过的话都能写入长期记忆吗？** 不能，只保存有持续价值且符合用户预期的信息，敏感内容需要更严格授权。
2. **模型推断出的偏好怎么处理？** 标记为推断和低置信，最好让用户确认，不能与明确声明等价。
