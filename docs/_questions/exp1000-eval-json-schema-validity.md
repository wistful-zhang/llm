## 面试时怎么答

回答“结构化输出评测”时，先给出核心结论，再补一个能检查结论的例子，最后主动说明这条边界：自动修复可能掩盖模型退化，也不能擅自猜测高风险字段。

**可以这样答：**

> 至少检查语法、Schema、字段语义、跨字段约束和下游执行安全，分别统计修复率与最终成功率。例如，生成缺字段、类型错、枚举外和语义冲突样本测试解析器。需要注意的是，自动修复可能掩盖模型退化，也不能擅自猜测高风险字段。
