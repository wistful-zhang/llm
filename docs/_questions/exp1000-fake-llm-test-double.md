## 面试时怎么答

回答“确定性模型测试替身”时，先给出核心结论，再补一个能检查结论的例子，最后主动说明这条边界：Fake 通过只能证明应用逻辑，质量仍需真实模型离线评测。

**可以这样答：**

> 核心判断是，Fake 应按输入条件返回预设流、错误或工具调用，用来测试编排状态与异常，而非模拟真实语言能力。实际验证可采用这个办法：覆盖正常、超时、半截流、非法 JSON 和重复工具调用。此外，Fake 通过只能证明应用逻辑，质量仍需真实模型离线评测。
