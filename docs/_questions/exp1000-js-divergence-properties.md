**可以这样答：**

> JS 先引入两分布的平均分布，再计算双方到平均分布的 KL，因此对称且有界。一个直接的检查办法是：用它比较训练语料与线上请求的类别分布漂移。这个结论的边界是，有界不代表容易估计，分桶方式和样本稀疏仍会改变结论。
