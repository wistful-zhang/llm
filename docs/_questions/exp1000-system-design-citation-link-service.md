## 面试时怎么答

说明模型只输出内部证据 ID，由服务端解析、鉴权并生成短时链接。

**可以这样答：**

> 检索器为片段生成稳定证据 ID，模型只引用这些 ID，不能自行拼接存储 URL。引用服务根据当前用户重新检查文档与页级权限，再返回标题、页码、证据预览和短时签名链接。链接不能携带长期凭据，也不能通过递增 ID 枚举其他文档。文档更新或删除后旧引用显示版本状态或失效，而不是静默跳到内容不同的新版本。

## 常见追问

1. **分享答案给别人时引用怎么办？** 接收者按自己的权限重新解析，不能复用发送者的签名链接。
2. **引用文档移动路径后会断吗？** 使用逻辑文档与版本 ID，不依赖易变路径，展示层再解析当前位置。
