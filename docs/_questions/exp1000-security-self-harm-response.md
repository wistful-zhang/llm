## 面试时怎么答

先用自己的话说清“自伤场景响应”，再给出一项可检查的证据或例子；结尾别漏掉模型不能诊断或替代紧急服务，地区与语言差异必须更新。

**可以这样答：**

> 应评估同理表达、鼓励寻求即时现实支持、危急程度识别和避免具体伤害指导，并适配地区资源。一个直接的检查办法是：由专业 Rubric 覆盖含蓄、明确和第三人称场景。这个结论的边界是，模型不能诊断或替代紧急服务，地区与语言差异必须更新。
