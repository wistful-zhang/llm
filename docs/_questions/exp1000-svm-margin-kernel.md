## 面试时怎么答

组织答案时围绕“支持向量机”展开：把定义或机制讲清楚，用具体例子验证，并说明核矩阵计算随样本数增长，核和超参数选择也会明显影响结果。

**可以这样答：**

> 核心判断是，线性 SVM 通过最大化几何间隔提升鲁棒性，核函数用内积替代显式高维映射。实际验证可采用这个办法：在二维数据上标出支持向量并比较不同 C 的边界。此外，核矩阵计算随样本数增长，核和超参数选择也会明显影响结果。

## 常见追问

1. **不铺背景，直接说明“支持向量机”的核心机制或判断。** 线性 SVM 通过最大化几何间隔提升鲁棒性，核函数用内积替代显式高维映射
2. **把“支持向量机”落到一个可检查的例子，你会怎么做？** 在二维数据上标出支持向量并比较不同 C 的边界
3. **什么情况下不能直接套用“支持向量机”？** 核矩阵计算随样本数增长，核和超参数选择也会明显影响结果
