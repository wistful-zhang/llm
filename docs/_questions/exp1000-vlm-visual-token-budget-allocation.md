## 面试时怎么答

按任务所需细节动态分辨率，并为文本指令、证据和输出预留空间。

**可以这样答：**

> 视觉 Token 数由分辨率、切片和压缩模块共同决定，不能只看图片文件大小。OCR 和细粒度定位需要更多视觉 Token，粗粒度分类则可用较低分辨率。系统先为指令、历史和输出预留文本预算，再根据任务与图片复杂度分配剩余空间。超限时优先选择相关区域或分阶段观察，而不是在不提示的情况下截断图片。

## 常见追问

1. **多张图片应该平均分配 Token 吗？** 不一定，应按问题相关度、分辨率和信息密度分配，同时保证每张有最低可用表示。
2. **怎样知道视觉 Token 已经够用？** 画任务准确率随 Token 数的曲线，并按小字、密集目标等难例切片观察。
