## 面试时怎么答

口述“逐样本梯度向量化”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：数据相关控制流、状态突变和部分原地操作可能不兼容；向量化还会提高中间张量峰值，未必总比分块循环省显存。

**可以这样答：**

> 关键点是，先把模型调用写成无状态纯函数，用 grad 得到单样本梯度函数，再让 vmap 沿 Batch 维批量变换，从而把 Python 循环下沉为框架可组合的向量化算子。验证时可以这样做：对一个线性分类器计算每条样本的参数梯度，并与逐条循环结果和耗时核对。但数据相关控制流、状态突变和部分原地操作可能不兼容；向量化还会提高中间张量峰值，未必总比分块循环省显存。
