## 面试时怎么答

结合相同状态哈希、重复工具参数、无新增证据和硬预算回答。

**可以这样答：**

> 编排器可对规范化的计划状态、工具名和参数做哈希，连续出现相同组合时判为重复。还应跟踪每步是否新增证据、缩小未决问题或改变外部状态，只有措辞变化不算进展。达到重复阈值时停止当前路径，尝试一次预定义替代策略或向用户澄清。无论检测多智能，仍要有最大步数和费用硬上限。

## 常见追问

1. **相同工具不同分页算重复吗？** 不算，游标变化表示有进展，但也要受总页数和结果增量限制。
2. **模型声称下一步会成功要继续吗？** 不能仅凭声明，应根据新参数、证据或策略是否实质变化判断。
