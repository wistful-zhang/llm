## 面试时怎么答

比较词表覆盖、位值对齐、序列长度和未见数字泛化。

**可以这样答：**

> 整串数字作为 Token 很省长度，但词表不可能覆盖所有数，稀有数字表示不稳定。逐位切分泛化开放、位值清楚，却让长数字序列变长，并要求模型跨多步组合。固定三位等分块是折中方案，但从左还是从右对齐会影响小数、千分位和算术进位，最终应按目标任务实测。

## 常见追问

1. **为什么右对齐分块可能利于整数运算？** 个位和千位边界在不同长度数字间更一致，便于学习位值和进位。
2. **科学计数法需要特殊处理吗？** 最好保证符号、指数标记和数字片段有稳定切分，否则同一数值格式差异过大。
