## 面试时怎么答

口述“训练数据投毒”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：高质量但恶意的语义投毒难靠统计过滤发现，来源和权限治理更关键。

**可以这样答：**

> 关键点是，攻击者可注入错误事实、偏见或触发行为，应通过来源信誉、异常聚类、去重、数据版本和小规模消融检测。验证时可以这样做：在受控数据中注入少量异常模式，比较训练前后行为和样本影响。但高质量但恶意的语义投毒难靠统计过滤发现，来源和权限治理更关键。
