## 面试时怎么答

口述“数据删除传播”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：不可变审计与法定留存需单独定义，不能承诺无法实现的即时物理删除。

**可以这样答：**

> 建立数据血缘，让删除覆盖原始记录、索引、缓存、派生特征和允许范围内的备份过期流程。一个直接的检查办法是：用唯一 canary 追踪删除前后各存储和检索结果。这个结论的边界是，不可变审计与法定留存需单独定义，不能承诺无法实现的即时物理删除。

## 常见追问

1. **请用自己的话说明“数据删除传播”的核心做法。** 建立数据血缘，让删除覆盖原始记录、索引、缓存、派生特征和允许范围内的备份过期流程
2. **你准备怎样举例证明自己理解“数据删除传播”？** 用唯一 canary 追踪删除前后各存储和检索结果
3. **使用“数据删除传播”前还要确认什么？** 不可变审计与法定留存需单独定义，不能承诺无法实现的即时物理删除
