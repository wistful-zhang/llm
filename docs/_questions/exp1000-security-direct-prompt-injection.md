## 面试时怎么答

组织答案时围绕“直接提示注入”展开：把定义或机制讲清楚，用具体例子验证，并说明自然语言规则无法提供绝对隔离，高风险能力必须在模型外授权。

**可以这样答：**

> 核心判断是，攻击者在用户输入中要求覆盖高优先级规则，防线需依靠指令分层、能力限制和输出验证而非秘密提示词。实际验证可采用这个办法：覆盖角色伪装、规则冲突、编码和多轮诱导，并验证系统仍拒绝越权动作。此外，自然语言规则无法提供绝对隔离，高风险能力必须在模型外授权。

## 常见追问

1. **不铺背景，直接说明“直接提示注入”的核心机制或判断。** 攻击者在用户输入中要求覆盖高优先级规则，防线需依靠指令分层、能力限制和输出验证而非秘密提示词

3. **什么情况下不能直接套用“直接提示注入”？** 自然语言规则无法提供绝对隔离，高风险能力必须在模型外授权
