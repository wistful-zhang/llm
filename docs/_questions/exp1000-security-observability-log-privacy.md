## 面试时怎么答

这题不要只报术语。先解释“隐私友好的日志”的核心逻辑，再说明如何验证，最后指出脱敏后的组合字段仍可能重新识别个人，访问审计同样必要。

**可以这样答：**

> 默认记录结构化元数据、哈希和错误类别，正文按授权采样、脱敏、加密并设置短保留期。一个直接的检查办法是：用敏感样本检查日志、Trace、告警和导出链路是否泄漏。这个结论的边界是，脱敏后的组合字段仍可能重新识别个人，访问审计同样必要。
