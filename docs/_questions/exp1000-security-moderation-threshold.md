## 面试时怎么答

组织答案时围绕“审核阈值分级”展开：把定义或机制讲清楚，用具体例子验证，并说明统一阈值会伤害低基准率类别，分布漂移也需持续重校准。

**可以这样答：**

> 不同风险类别的误放与误拦代价不同，应分别校准阈值，并为不确定高风险样本设置人工队列。例如，按类别画 PR 曲线并模拟审核量与漏放成本。需要注意的是，统一阈值会伤害低基准率类别，分布漂移也需持续重校准。

## 常见追问

1. **不铺背景，直接说明“审核阈值分级”的核心机制或判断。** 不同风险类别的误放与误拦代价不同，应分别校准阈值，并为不确定高风险样本设置人工队列
2. **把“审核阈值分级”落到一个可检查的例子，你会怎么做？** 按类别画 PR 曲线并模拟审核量与漏放成本
3. **什么情况下不能直接套用“审核阈值分级”？** 统一阈值会伤害低基准率类别，分布漂移也需持续重校准
