## 面试时怎么答

口述“N-gram 平滑”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：高阶 N-gram 参数爆炸且上下文有限，不适合表示长距离语义。

**可以这样答：**

> 有限语料会让未见组合概率为零，平滑把部分概率质量分配给未见事件并配合回退。具体例子是，统计二元词频，比较加一平滑与插值后的句子概率。真正落地前还要检查，高阶 N-gram 参数爆炸且上下文有限，不适合表示长距离语义。

## 常见追问

1. **请用自己的话说明“N-gram 平滑”的核心做法。** 有限语料会让未见组合概率为零，平滑把部分概率质量分配给未见事件并配合回退
2. **你准备怎样举例证明自己理解“N-gram 平滑”？** 统计二元词频，比较加一平滑与插值后的句子概率
3. **使用“N-gram 平滑”前还要确认什么？** 高阶 N-gram 参数爆炸且上下文有限，不适合表示长距离语义
