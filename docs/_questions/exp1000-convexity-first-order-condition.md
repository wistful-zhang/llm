## 面试时怎么答

建议按“结论—例子—边界”回答“凸性与全局最优”：核心判断要明确，验证要可复现，并说明深度网络通常非凸，一阶条件只能说明驻点而非全局最优。

**可以这样答：**

> 可微凸函数始终位于任一点切平面之上，因此梯度为零的点不会存在更低函数值。要验证这一点，可以采用这个办法：用二次函数和非凸双井函数比较驻点。使用时不能忽略，深度网络通常非凸，一阶条件只能说明驻点而非全局最优。
