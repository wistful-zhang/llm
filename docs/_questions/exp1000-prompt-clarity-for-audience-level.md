## 面试时怎么答

把受众差异落到先验知识、术语密度、例子和深度，不要只改语气。

**可以这样答：**

> Prompt 应明确受众已经掌握什么、希望完成什么以及可接受的术语和数学深度。新手版优先直觉、最小术语和具体例子，工程师版强调接口、边界和实践，专家版再展开假设、复杂度和争议。核心事实应保持一致，不能为了通俗而改变结论。可以让用户选择层级，并提供“继续深入”而不是一次生成三份长答案。

## 常见追问

1. **怎样判断用户是新手还是专家？** 优先让用户选择或根据明确任务推断，不要仅凭语言风格做敏感画像。
2. **新手回答能省略例外吗？** 可以减少细节，但影响结论的重要边界仍要用简单语言说明。
