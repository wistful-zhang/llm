## 面试时怎么答

建议按“结论—例子—边界”回答“LLM 安全事故响应”：核心判断要明确，验证要可复现，并说明删除单条输出或改提示词通常不足，取证过程也必须保护隐私。

**可以这样答：**

> 先停止高风险能力、撤销凭据并保全日志，再确认影响范围、通知责任方、修复根因和补回归测试。一个直接的检查办法是：演练泄漏 canary、异常工具调用和跨租户访问三类事件。这个结论的边界是，删除单条输出或改提示词通常不足，取证过程也必须保护隐私。
