## 面试时怎么答

把删除建模为跨存储流程，覆盖派生数据、缓存、备份策略和完成证明。

**可以这样答：**

> 先用稳定主体和记忆 ID 找到原始记录、向量、摘要、缓存以及从它派生的长期状态。在线副本应立即不可检索，后台执行幂等删除并通过清单对账。不可即时修改的备份按明确保留周期过期，恢复时必须重放删除 tombstone，防止数据复活。向用户返回删除范围和状态，但审计记录只保留必要的非敏感证明。

## 常见追问

1. **删除对话就等于删除记忆吗？** 不等于，结构化画像和向量可能在独立系统，需要统一数据血缘。
2. **模型已经训练过该数据怎么办？** 在线记忆删除与模型训练治理不同，应按数据来源和适用法规采用隔离、再训练或其他流程。
