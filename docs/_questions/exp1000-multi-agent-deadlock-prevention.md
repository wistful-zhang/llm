## 面试时怎么答

使用有向依赖图、超时、所有权和禁止循环等待回答。

**可以这样答：**

> 协调器应维护任务依赖图，在新增委派前检测环路，不能只靠 Agent 自己描述等待关系。每个子任务有唯一 Owner、截止时间和完成协议，等待超时后进入取消、替代或人工升级。共享资源采用统一锁顺序或无锁消息传递，减少循环占用。监控要能展示谁在等待谁，并在图不再推进时自动终止，而不是继续消耗模型调用。

## 常见追问

1. **两个 Agent 对话协商能解决吗？** 可能解决简单冲突，但没有外部依赖状态仍可能无限互相询问。
2. **怎样检测活锁？** 状态持续变化但没有任务完成或新证据时也应判为无进展并触发停止。
