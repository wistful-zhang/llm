## 面试时怎么答

这题不要只报术语。先解释“供应商凭据管理”的核心逻辑，再说明如何验证，最后指出环境变量也会被进程或诊断工具读取，轮换必须覆盖缓存和长进程。

**可以这样答：**

> 凭据放入专用 Secret Manager，运行时按最小权限获取，日志不展示，并支持双密钥重叠轮换。具体例子是，演练旧密钥撤销、单供应商泄漏和紧急切换。真正落地前还要检查，环境变量也会被进程或诊断工具读取，轮换必须覆盖缓存和长进程。
