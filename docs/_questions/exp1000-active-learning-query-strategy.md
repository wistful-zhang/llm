## 面试时怎么答

这题不要只报术语。先解释“主动学习采样”的核心逻辑，再说明如何验证，最后指出模型初期置信不可靠，线上抽样还要防止分布和公平性偏差。

**可以这样答：**

> 可以从这条主线理解：只选不确定样本可能集中于异常点，应联合不确定性、多样性、代表性和标注成本。再用一个最小验证说明：比较随机、不确定性和聚类覆盖采样的学习曲线。最后明确限制：模型初期置信不可靠，线上抽样还要防止分布和公平性偏差。

## 常见追问

1. **面试官要求一句话概括“主动学习采样”时，你怎么说？** 只选不确定样本可能集中于异常点，应联合不确定性、多样性、代表性和标注成本
2. **你会怎样用数据、代码或手算验证“主动学习采样”？** 比较随机、不确定性和聚类覆盖采样的学习曲线
3. **回答“主动学习采样”时必须主动补充哪项限制？** 模型初期置信不可靠，线上抽样还要防止分布和公平性偏差
