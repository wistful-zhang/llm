## 面试时怎么答

说明证据 ID、主张拆分、引用校验和展示层映射。

**可以这样答：**

> 检索片段要带不可由模型伪造的内部证据 ID，并在 Prompt 中要求每个外部事实句标注支持它的 ID。生成后检查 ID 是否存在、用户是否有权限访问，以及对应片段是否蕴含该句。复合句包含多个主张时应拆开或分别引用，不能用一个来源覆盖全部。展示层再把内部 ID 映射为文档标题、页码和安全链接。

## 常见追问

1. **常识性内容也需要引用吗？** 取决于产品目标，关键业务事实必须引用，解释性常识可按策略省略但不能冒充来源内容。
2. **同一证据被多句引用有问题吗？** 没有，只要它确实支持每个主张；重复引用可在界面上合并展示。
