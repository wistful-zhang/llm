## 面试时怎么答

先用自己的话说清“完整响应缓存”，再给出一项可检查的证据或例子；结尾别漏掉个性化、时效性和随机生成会降低可缓存性，敏感输出不能跨用户共享。

**可以这样答：**

> 核心判断是，只有规范化输入、模型版本、参数、权限和外部依赖都进入缓存键，且业务允许复用时才安全。实际验证可采用这个办法：对相同 Prompt 不同租户、温度和知识版本验证是否错误命中。此外，个性化、时效性和随机生成会降低可缓存性，敏感输出不能跨用户共享。
