## 面试时怎么答

提供目标、已验证事实、执行状态和待决问题，避免倾倒整段隐私对话。

**可以这样答：**

> 交接包应包含用户目标、当前状态、已完成动作、失败原因、待确认问题和可追溯证据。明确哪些内容是模型推断、哪些来自工具或用户确认，防止人工把猜测当事实。只包含处理该事件必要的敏感数据，并遵循客服或专家的访问权限。人工处理结果写回结构化状态，让 Agent 后续能继续而不是重复询问。

## 常见追问

1. **要把完整聊天记录都给人工吗？** 通常不需要，可提供受控展开入口，默认先展示最小相关摘要和证据。
2. **人工多久没处理怎么办？** 设置 SLA、提醒和升级队列，同时向用户同步等待状态与可选取消。
