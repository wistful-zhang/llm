## 面试时怎么答

回答“随机森林去相关”时，先给出核心结论，再补一个能检查结论的例子，最后主动说明这条边界：高度不平衡、强时序依赖或需要外推时仍可能表现不佳。

**可以这样答：**

> 这件事可以概括为：Bootstrap 样本与随机特征子集让各棵树错误相关性下降，聚合后降低方差。落到实验或实现上，比较单树、只采样样本和同时采样特征的验证波动。同时要确认，高度不平衡、强时序依赖或需要外推时仍可能表现不佳。

## 常见追问

1. **请把“随机森林去相关”的核心结论压缩成一句话。** Bootstrap 样本与随机特征子集让各棵树错误相关性下降，聚合后降低方差
2. **你会用什么例子或检查验证“随机森林去相关”？** 比较单树、只采样样本和同时采样特征的验证波动
3. **“随机森林去相关”最重要的适用边界是什么？** 高度不平衡、强时序依赖或需要外推时仍可能表现不佳
