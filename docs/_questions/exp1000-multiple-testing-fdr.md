## 面试时怎么答

这题不要只报术语。先解释“多重比较校正”的核心逻辑，再说明如何验证，最后指出事后挑选指标和切片仍会造成选择偏差，校正不能修复坏实验设计。

**可以这样答：**

> 核心判断是，检验数量增加会提高至少一次假阳性的概率，可控制家族错误率或错误发现率。实际验证可采用这个办法：对多个模型和数据切片预注册比较，采用 Holm 或 Benjamini-Hochberg 校正。此外，事后挑选指标和切片仍会造成选择偏差，校正不能修复坏实验设计。
