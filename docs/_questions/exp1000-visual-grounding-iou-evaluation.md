## 面试时怎么答

解释 IoU 阈值的局限，并补充点定位、召回、多目标和语义正确性。

**可以这样答：**

> IoU 衡量预测框与真值框的重叠，但对小目标非常敏感，也不能判断模型是否理解了正确类别。应同时报告不同 IoU 阈值下准确率、中心点命中、目标尺寸切片和多目标 Recall。短语包含关系或属性时，还要检查选中的对象是否满足语义，而不只是碰到附近区域。标注框本身可能有主观差异，因此应记录标注规范和人工上限。

## 常见追问

1. **小目标 IoU 很低怎么办？** 可补充归一化中心距离或点命中指标，但不能因此完全放弃边界质量。
2. **分割掩码是否总比框好？** 更精细但标注成本更高，任务只需点击或粗定位时框可能足够。
