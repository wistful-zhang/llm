## 面试时怎么答

这类题不要套用别人的经历。围绕“自建与采购决策”讲清背景、目标、本人动作与真实证据，最后说明不能只比较单次 token 单价，也不要虚构合同和采购信息。

**可以这样答：**

> 比较数据边界、峰值容量、团队能力、单位经济、功能控制、SLA 和迁移成本。证据部分只写本人真实材料：替换为本人项目的流量区间、合规约束和总拥有成本估算。还要明确，不能只比较单次 token 单价，也不要虚构合同和采购信息。

## 常见追问

1. **回答“自建与采购决策”时应覆盖哪些本人事实？** 比较数据边界、峰值容量、团队能力、单位经济、功能控制、SLA 和迁移成本
2. **可以用什么真实证据支撑“自建与采购决策”？** 替换为本人项目的流量区间、合规约束和总拥有成本估算
3. **复盘“自建与采购决策”时必须主动承认哪项边界？** 不能只比较单次 token 单价，也不要虚构合同和采购信息
