## 面试时怎么答

这类题不要套用别人的经历。围绕“带教与评审”讲清背景、目标、本人动作与真实证据，最后说明不能声称改变他人表现而没有反馈，也不要把评审变成人身评价。

**可以这样答：**

> 使用可解释标准区分正确性、安全、性能和风格，先问设计意图，再给可执行建议并跟踪共同改进。证据部分只写本人真实材料：替换为本人真实评审案例、模板或新人独立交付结果。还要明确，不能声称改变他人表现而没有反馈，也不要把评审变成人身评价。

## 常见追问

1. **回答“带教与评审”时应覆盖哪些本人事实？** 使用可解释标准区分正确性、安全、性能和风格，先问设计意图，再给可执行建议并跟踪共同改进
2. **可以用什么真实证据支撑“带教与评审”？** 替换为本人真实评审案例、模板或新人独立交付结果
3. **复盘“带教与评审”时必须主动承认哪项边界？** 不能声称改变他人表现而没有反馈，也不要把评审变成人身评价
