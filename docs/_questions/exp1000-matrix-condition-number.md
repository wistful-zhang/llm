## 面试时怎么答

建议按“结论—例子—边界”回答“矩阵条件数”：核心判断要明确，验证要可复现，并说明条件数描述问题敏感性，不等于具体算法一定不稳定。

**可以这样答：**

> 条件数衡量输入微小扰动被解放大的程度，奇异值最大值与最小值之比越大越病态。一个直接的检查办法是：对近共线特征的线性系统加入微小噪声并观察解变化。这个结论的边界是，条件数描述问题敏感性，不等于具体算法一定不稳定。

2. **哪项具体检查可以支撑你对“矩阵条件数”的判断？** 对近共线特征的线性系统加入微小噪声并观察解变化
3. **“矩阵条件数”最容易被忽略的前提是什么？** 条件数描述问题敏感性，不等于具体算法一定不稳定
