## 面试时怎么答

区分语法类型校验、跨字段语义和当前资源状态，并举出范围与权限例子。

**可以这样答：**

> Schema 只能保证字段类型、必填和基础格式，无法覆盖所有业务语义。日期区间可能前后颠倒，订单可能已关闭，收件人也可能不属于当前租户。执行层要做跨字段约束、资源存在性、权限、幂等和状态机校验。失败时返回结构化错误让 Agent 修正可修参数，而不是把数据库异常原样暴露。

## 常见追问

1. **业务校验应放工具内部还是 Agent 编排层？** 最终强制校验必须在工具内部，编排层可提前检查以改善体验。
2. **模型修正参数能重试几次？** 按错误类型设置有限次数，权限或不可变状态错误不应反复重试。
