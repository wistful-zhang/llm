## 面试时怎么答

建议按“结论—例子—边界”回答“L1 与 L2 正则”：核心判断要明确，验证要可复现，并说明特征尺度必须一致，稀疏不等于选中了因果或稳定特征。

**可以这样答：**

> 核心判断是，L1 的尖角约束更容易把参数压到零，L2 平滑收缩大参数但通常不产生精确稀疏。实际验证可采用这个办法：在相关特征回归中比较系数路径和验证误差。此外，特征尺度必须一致，稀疏不等于选中了因果或稳定特征。

2. **哪项具体检查可以支撑你对“L1 与 L2 正则”的判断？** 在相关特征回归中比较系数路径和验证误差
