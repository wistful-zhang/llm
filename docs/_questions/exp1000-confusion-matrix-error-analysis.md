## 面试时怎么答

回答“分类错误分析”时，先给出核心结论，再补一个能检查结论的例子，最后主动说明这条边界：汇总矩阵会隐藏长尾群体和样本难度，不能只优化最大格子。

**可以这样答：**

> 核心判断是，先看主要混淆对及其支持度，再抽样阅读并按数据、标签、模型和阈值原因归因。实际验证可采用这个办法：为最大混淆对建立可复现切片并测试改动前后。此外，汇总矩阵会隐藏长尾群体和样本难度，不能只优化最大格子。
