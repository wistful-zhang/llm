## 面试时怎么答

定义相关、部分相关和直接支持的区别，并加入双人标注与分歧仲裁。

**可以这样答：**

> 标注指南要区分主题相关、包含部分答案和足以直接支持答案，不能把它们都记为相关。对时间、权限和版本条件给出明确规则，同一文本在不同 Query 下可以有不同标签。使用多名标注者覆盖一部分样本，计算一致性并对分歧做领域仲裁。保存判断依据和关键证据跨度，后续才能分析是检索失败、切分失败还是标签问题。

## 常见追问

1. **只有一个正确文档时怎么标？** 仍应标记其他候选的部分相关程度，否则无法评估排序质量和假阳性。
2. **可以让 LLM 自动标注吗？** 可用于预标和扩量，但需用人工金标准校准，并对高分歧样本复核。
