## 面试时怎么答

建议按“结论—例子—边界”回答“自适应越狱评测”：核心判断要明确，验证要可复现，并说明自动 Judge 可能被攻击文本一起欺骗，高风险样本需人工复核。

**可以这样答：**

> 这件事可以概括为：攻击者会根据拒答反馈迭代改写，固定题只测已知模板，应加入多轮、自适应搜索和新攻击留出集。落到实验或实现上，让攻击器在查询预算内优化并报告攻击成功率随预算变化。同时要确认，自动 Judge 可能被攻击文本一起欺骗，高风险样本需人工复核。
