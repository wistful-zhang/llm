## 面试时怎么答

建议按“结论—例子—边界”回答“版权内容复现风险”：核心判断要明确，验证要可复现，并说明公开可访问不等于可任意复制，具体法律判断需专业意见。

**可以这样答：**

> 关键点是，评测应关注用户是否定位特定作品、输出与原文重合长度及是否可替代原作，并优先提供概述。验证时可以这样做：对公开作品片段做不同提示强度的匹配测试并记录最长重合。但公开可访问不等于可任意复制，具体法律判断需专业意见。
