## 面试时怎么答

建议按“结论—例子—边界”回答“API 响应向前兼容”：核心判断要明确，验证要可复现，并说明宽容读取不等于接受任意数据，安全决策不能对未知值默认放行。

**可以这样答：**

> 这件事可以概括为：解析器应忽略并保留未知可选字段，对未知事件或枚举进入明确的安全分支，同时继续严格校验必需字段、类型和权限相关数据。落到实验或实现上，用录制响应加入未知字段、未知事件和缺失必填项，验证旧客户端能区分可忽略扩展与破坏性变更。同时要确认，宽容读取不等于接受任意数据，安全决策不能对未知值默认放行。

## 常见追问

1. **“API 响应向前兼容”最需要讲清的核心内容是什么？** 解析器应忽略并保留未知可选字段，对未知事件或枚举进入明确的安全分支，同时继续严格校验必需字段、类型和权限相关数据
2. **哪项具体检查可以支撑你对“API 响应向前兼容”的判断？** 用录制响应加入未知字段、未知事件和缺失必填项，验证旧客户端能区分可忽略扩展与破坏性变更
3. **“API 响应向前兼容”最容易被忽略的前提是什么？** 宽容读取不等于接受任意数据，安全决策不能对未知值默认放行
