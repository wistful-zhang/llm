## 面试时怎么答

要求工具注册时显式声明效果，并指出接口名叫 get 也可能触发状态变化。

**可以这样答：**

> 工具注册表应显式标注读、写、外部通信、费用和不可逆等级，不能根据 HTTP 方法或名称猜测。只读工具也要检查是否会刷新状态、触发计费或泄露敏感数据。有副作用的调用进入更严格的授权、确认和审计流程，预览与提交最好拆成两个工具。运行时发现工具实际行为与声明不符，应阻断并视为接口安全缺陷。

## 常见追问

1. **发送测试邮件算低风险吗？** 仍是外部通信，可能泄露内容或打扰他人，至少需要明确目标与环境限制。
2. **删除操作如何设计预览？** 先返回将删除的对象、数量和不可恢复影响，确认后携带预览版本执行。
