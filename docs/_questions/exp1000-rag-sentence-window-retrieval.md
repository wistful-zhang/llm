## 面试时怎么答

说明句子级索引与相邻上下文扩展，指出表格和代码等结构不适合机械按句切。

**可以这样答：**

> 它用单句或很短片段做精确召回，命中后扩展前后若干句供生成，适合连续叙述的说明文档。窗口太小会缺条件，太大又会带入别的主题，应根据段落边界和代词指代动态调整。表格、代码和列表不能简单按句号处理，需要专用解析。评测应关注命中句是否完整支持答案，以及扩展后无关 Token 比例。

## 常见追问

1. **窗口固定前后两句可以吗？** 可作基线，但跨段落时应停止，定义或条件句可能需要更大的同段窗口。
2. **索引文本和返回文本必须一样吗？** 不必，索引可用短句，返回可携带更完整上下文和元数据。
