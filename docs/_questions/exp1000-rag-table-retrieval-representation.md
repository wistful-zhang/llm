## 面试时怎么答

回答表头继承、行列语义、结构化查询与文本检索结合。

**可以这样答：**

> 表格不能只按字符切块，否则数据会失去表头、单位和行列关系。可把每行或逻辑分组序列化为带表名、列名和单位的文本，同时保留原始单元格结构。精确聚合和筛选优先转换为数据库查询，语义检索用于找到相关表和字段。返回模型时只提供所需行列并标明来源，数值计算交给代码完成。

## 常见追问

1. **合并单元格怎么处理？** 解析时把上层表头展开到具体列，并保留层级关系，不能简单留空。
2. **表格向量化一整张可以吗？** 小表可作补充，大表会主题混杂且超长，通常要按语义行组拆分。
