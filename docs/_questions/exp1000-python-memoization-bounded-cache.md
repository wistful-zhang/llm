## 面试时怎么答

这题不要只报术语。先解释“有界函数缓存”的核心逻辑，再说明如何验证，最后指出有副作用、依赖时间或返回可变对象的函数不适合直接缓存。

**可以这样答：**

> 把可哈希参数作为键，命中直接返回；使用 LRU 等策略限制条目，并为并发访问加同步。要验证这一点，可以采用这个办法：对昂贵纯函数统计命中率、淘汰顺序和多线程重复计算。使用时不能忽略，有副作用、依赖时间或返回可变对象的函数不适合直接缓存。

## 常见追问

1. **面试官要求一句话概括“有界函数缓存”时，你怎么说？** 把可哈希参数作为键，命中直接返回；使用 LRU 等策略限制条目，并为并发访问加同步
2. **你会怎样用数据、代码或手算验证“有界函数缓存”？** 对昂贵纯函数统计命中率、淘汰顺序和多线程重复计算
3. **回答“有界函数缓存”时必须主动补充哪项限制？** 有副作用、依赖时间或返回可变对象的函数不适合直接缓存
