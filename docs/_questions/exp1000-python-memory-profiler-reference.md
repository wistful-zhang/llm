## 面试时怎么答

这题不要只报术语。先解释“Python 内存增长排查”的核心逻辑，再说明如何验证，最后指出RSS 不下降不一定泄漏，GPU 与原生内存也不在普通堆快照内。

**可以这样答：**

> 这件事可以概括为：比较对象数量、分配栈、引用链和 RSS，确认对象是否仍可达，再区分应用缓存、原生库和分配器不归还。落到实验或实现上，固定流量运行多轮快照，按类型和调用栈查看净增长。同时要确认，RSS 不下降不一定泄漏，GPU 与原生内存也不在普通堆快照内。
