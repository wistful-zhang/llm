## 面试时怎么答

先用自己的话说清“嵌套对象复制”，再给出一项可检查的证据或例子；结尾别漏掉深拷贝可能昂贵或不适合文件、锁和 GPU Tensor，优先使用不可变配置。

**可以这样答：**

> 浅拷贝只复制外层容器，内部可变对象仍共享；深拷贝递归复制并处理循环引用。要验证这一点，可以采用这个办法：复制含列表和字典的模型配置，修改内层值观察原对象。使用时不能忽略，深拷贝可能昂贵或不适合文件、锁和 GPU Tensor，优先使用不可变配置。

## 常见追问

1. **如果只保留一个要点，“嵌套对象复制”是什么？** 浅拷贝只复制外层容器，内部可变对象仍共享；深拷贝递归复制并处理循环引用
2. **给出一个可以复现或手工检查“嵌套对象复制”的办法。** 复制含列表和字典的模型配置，修改内层值观察原对象
3. **在哪种条件下，“嵌套对象复制”会失效或被误读？** 深拷贝可能昂贵或不适合文件、锁和 GPU Tensor，优先使用不可变配置
