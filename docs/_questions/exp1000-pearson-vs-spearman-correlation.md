## 面试时怎么答

组织答案时围绕“Pearson 与 Spearman 相关”展开：把定义或机制讲清楚，用具体例子验证，并说明两种相关都不代表因果，并列秩和小样本会影响 Spearman 的稳定性。

**可以这样答：**

> Pearson 衡量线性关系并使用原始数值，Spearman 对秩计算相关，更适合单调但非线性或受异常值影响的数据。例如，构造线性、单调曲线和含异常点的三组数据比较两个系数。需要注意的是，两种相关都不代表因果，并列秩和小样本会影响 Spearman 的稳定性。

## 常见追问

1. **不铺背景，直接说明“Pearson 与 Spearman 相关”的核心机制或判断。** Pearson 衡量线性关系并使用原始数值，Spearman 对秩计算相关，更适合单调但非线性或受异常值影响的数据
2. **把“Pearson 与 Spearman 相关”落到一个可检查的例子，你会怎么做？** 构造线性、单调曲线和含异常点的三组数据比较两个系数
3. **什么情况下不能直接套用“Pearson 与 Spearman 相关”？** 两种相关都不代表因果，并列秩和小样本会影响 Spearman 的稳定性
