## 面试时怎么答

说明字符正确不等于行列关系正确，并给出单元格网格与合并表头处理。

**可以这样答：**

> 普通 OCR 主要识别字符，而表格问答还必须恢复单元格、行列、合并表头和单位对应关系。字符全对但错位到另一列，答案仍会完全错误。系统应先检测表格结构并生成带坐标的单元格网格，再把查询转换为筛选和计算。复杂表格保留原图区域供模型复核，数值运算由确定性代码完成。

## 常见追问

1. **无边框表格怎么识别？** 结合文字对齐、间距和语义表头推断结构，置信不足时保留多个候选布局。
2. **跨页表格如何合并？** 利用重复表头、列宽和文档顺序对齐，同时验证页间单位和列定义没有变化。
