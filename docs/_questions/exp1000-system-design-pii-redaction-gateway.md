## 面试时怎么答

覆盖规则与模型检测、稳定占位符、可逆映射隔离、输出复原和漏检监控。

**可以这样答：**

> 网关按数据分类使用规则、词典和实体模型检测姓名、联系方式、证件与业务标识，并依据用途决定删除、不可逆遮挡或令牌化。占位符在一次请求内稳定，保持实体关系；可逆映射只存在受控服务，外部模型看不到。模型输出回流后检查是否出现新 PII，再由授权服务按允许字段复原。系统用带真值的多语言样本测漏检与误杀，原文日志默认关闭或严格限权。

## 常见追问

1. **所有数字都遮挡可以吗？** 会严重破坏任务，应按上下文识别类型，并对金额、日期等非身份数字保留。
2. **模型需要区分两个同名人怎么办？** 使用不同稳定实体占位符和非敏感关系信息，不必暴露真实姓名。
