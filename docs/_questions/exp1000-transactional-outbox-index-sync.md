## 面试时怎么答

**可以这样答：**

> 业务数据与待发布事件在同一数据库事务写入，后台可靠投递到索引并用事件 ID 幂等消费。一个直接的检查办法是：在提交后、投递前和索引写入后分别崩溃，验证最终状态与重复处理。这个结论的边界是，Outbox 提供最终一致而非原子跨系统可见，事件顺序和删除仍需版本控制。
