## 面试时怎么答

口述“哈希表查找”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：键必须可哈希且相等对象需有相同哈希，可变内容不应参与哈希。

**可以这样答：**

> dict 用哈希定位槽位并处理冲突，负载受控时平均常数时间；大量冲突或恶意键会增加探测成本。要验证这一点，可以采用这个办法：自定义具有相同 hash 的键并测量查找随数量变化。使用时不能忽略，键必须可哈希且相等对象需有相同哈希，可变内容不应参与哈希。
