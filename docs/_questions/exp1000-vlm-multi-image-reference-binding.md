## 面试时怎么答

说明稳定图像编号、消息顺序、缩略图确认和跨图推理的证据绑定。

**可以这样答：**

> 每张图应有稳定内部 ID，并在界面和模型上下文中使用一致的显示顺序与标签。用户新增、删除或重排图片后要重新确认引用，不能依赖上传文件名猜测。模型输出涉及图片事实时附图像 ID 和区域信息，便于展示层核对。指代仍不清楚时应展示缩略图让用户选择，而不是默认第二张就是某个附件。

## 常见追问

1. **可以让模型自己给图片命名吗？** 可生成描述性别名帮助阅读，但内部绑定仍用不可变 ID，别名不能作为唯一主键。
2. **多轮对话中图片顺序变化怎么办？** 保留会话级 ID，并在每轮明确当前可见图片映射，避免沿用旧序号。
