## 面试时怎么答

指出模型采样、外部世界和时间变化，并说明录制响应与模拟重放的边界。

**可以这样答：**

> 模型输出可能受采样和服务实现影响，搜索、数据库和网页内容也会随时间变化。真实重放若再次调用副作用工具，还可能造成重复操作。调试时应冻结模型与 Prompt 版本，记录工具响应、时间和随机参数，用模拟工具回放已有结果。这样可以复现编排逻辑，但无法保证重现当时外部世界的全部状态，应明确区分模拟与现场重跑。

## 常见追问

1. **保存随机种子就够了吗？** 不够，底层推理可能非确定，工具和数据状态也会变化。
2. **什么时候允许真实重跑？** 仅限隔离测试环境或确认所有工具只读且数据快照固定的情况。
