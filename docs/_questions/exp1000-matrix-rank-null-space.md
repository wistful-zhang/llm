## 面试时怎么答

组织答案时围绕“矩阵秩和零空间”展开：把定义或机制讲清楚，用具体例子验证，并说明数值秩依赖阈值，近奇异矩阵在浮点计算中可能表现为降秩。

**可以这样答：**

> 核心判断是，秩是独立方向数量，零空间包含映射后变为零的方向，因此非平凡零空间意味着信息不可逆丢失。实际验证可采用这个办法：构造低秩投影并找出被压掉的输入方向。此外，数值秩依赖阈值，近奇异矩阵在浮点计算中可能表现为降秩。
