## 面试时怎么答

强调秘密不进入模型上下文，由专用凭据服务引用，并对输出和日志脱敏。

**可以这样答：**

> 秘密值应由凭据服务托管，工作流只传递不可读引用，真正调用时由可信执行器注入。Prompt、工具描述、轨迹日志和错误信息都不能包含明文密钥。用户粘贴秘密时应提示改用安全连接流程，并尽快从会话与日志中脱敏。验证码属于短期认证因素，不能让 Agent 保存或在未确认的动作中复用。

## 常见追问

1. **模型需要密钥才能生成命令怎么办？** 命令使用占位符，执行器在受控环境注入，模型不需要知道真实值。
2. **哈希后的密钥可以记录吗？** 可记录不可逆指纹用于关联，但要使用安全哈希并评估低熵秘密的枚举风险。
