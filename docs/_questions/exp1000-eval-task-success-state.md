## 面试时怎么答

这题不要只报术语。先解释“任务完成评测”的核心逻辑，再说明如何验证，最后指出模拟器与真实业务可能有差距，部分完成和用户取消要单独定义。

**可以这样答：**

> 可以从这条主线理解：不要只评最后一句话，要核对外部状态是否达到目标、约束是否满足且没有不允许的副作用。再用一个最小验证说明：在订票模拟器中检查订单状态、日期、价格和确认步骤。最后明确限制：模拟器与真实业务可能有差距，部分完成和用户取消要单独定义。

## 常见追问

1. **面试官要求一句话概括“任务完成评测”时，你怎么说？** 不要只评最后一句话，要核对外部状态是否达到目标、约束是否满足且没有不允许的副作用
2. **你会怎样用数据、代码或手算验证“任务完成评测”？** 在订票模拟器中检查订单状态、日期、价格和确认步骤
