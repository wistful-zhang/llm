## 面试时怎么答

从租户策略、数据分类、区域能力和不可用降级回答，不能只依赖用户 IP。

**可以这样答：**

> 租户配置和数据分类共同决定允许处理与存储的区域，用户 IP 只能辅助选择最近入口，不能作为合规依据。网关把请求路由到获准区域内的模型、日志、向量库和工具，任何跨区依赖都要在架构清单中可见。目标区域不可用时按策略失败或使用合规备区，不能为了可用性偷偷跨境。配置变更经过审批，并用合成探针验证实际数据路径。

## 常见追问

1. **模型供应商说不训练数据就满足驻留吗？** 不等于，仍要确认实际推理、日志、备份和支持访问所在区域。
2. **全球账号的会话怎么同步？** 只同步允许跨区的最小元数据，敏感会话留在归属区域，通过受控代理访问。
