## 面试时怎么答

组织答案时围绕“Bernoulli 与 Binomial 分布”展开：把定义或机制讲清楚，用具体例子验证，并说明样本若相关、难度不均或重复，就不满足简单独立同分布假设。

**可以这样答：**

> 单次成败可建模为 Bernoulli，多次独立同分布成功次数服从 Binomial。例如，把每道题是否答对记为零一变量，估计准确率及标准误。需要注意的是，样本若相关、难度不均或重复，就不满足简单独立同分布假设。

## 常见追问

1. **不铺背景，直接说明“Bernoulli 与 Binomial 分布”的核心机制或判断。** 单次成败可建模为 Bernoulli，多次独立同分布成功次数服从 Binomial
2. **把“Bernoulli 与 Binomial 分布”落到一个可检查的例子，你会怎么做？** 把每道题是否答对记为零一变量，估计准确率及标准误
3. **什么情况下不能直接套用“Bernoulli 与 Binomial 分布”？** 样本若相关、难度不均或重复，就不满足简单独立同分布假设
