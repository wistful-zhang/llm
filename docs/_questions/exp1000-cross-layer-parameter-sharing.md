## 面试时怎么答

区分参数量、计算量和激活量，避免把共享权重误说成少算几层。

**可以这样答：**

> 跨层共享会减少权重参数和加载带宽，但每层计算仍然执行，所以 FLOPs 和大部分激活并不会同比下降。重复使用同一变换形成了正则化，也限制不同深度的功能分工，质量可能需要靠加宽或增加步数补偿。部署端若权重能留在缓存中可能受益，但实际速度还取决于内核与显存层级。

## 常见追问

1. **只共享 FFN 和共享整层有什么区别？** 只共享 FFN 保留各层注意力模式的自由，压缩程度较小但通常更灵活。
2. **共享权重的梯度怎样计算？** 所有使用位置产生的梯度会累加到同一参数上。
