## 面试时怎么答

比较可控性、韵律信息、延迟和可观测性，并给出场景化选择。

**可以这样答：**

> ASR 加文本 LLM 模块清晰、可单独调试和替换，也容易做敏感词、日志和文本检索。它会丢失语气、停顿和非语言声音，并累积识别误差。原生音频模型能保留更多声学信息并实现低延迟对话，但训练与评测更复杂，输出也更难审计。客服转写等准确文本优先的场景常用级联，情绪和实时交互重要时可考虑原生或混合方案。

## 常见追问

1. **级联误差怎么缓解？** 向 LLM 提供 ASR 候选、时间戳和置信度，对关键实体再做确认或专用纠错。
2. **原生音频模型就不需要 ASR 指标吗？** 仍可用转写作为诊断切片，但还要评估语义、延迟、说话人和韵律任务。
