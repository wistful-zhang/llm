## 面试时怎么答

口述“语义匹配评测”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：高相似文本可能事实相反，Embedding 分数不能替代任务规则。

**可以这样答：**

> 先用人工标注的等价与不等价答案校准相似度阈值，再按误报漏报代价选择工作点。要验证这一点，可以采用这个办法：构造数字、否定和实体仅差一个词的困难负例，画 ROC 与 PR 曲线。使用时不能忽略，高相似文本可能事实相反，Embedding 分数不能替代任务规则。
