## 面试时怎么答

建议按“结论—例子—边界”回答“延迟计时与时钟偏差”：核心判断要明确，验证要可复现，并说明单调时钟值不能跨进程直接相减，墙上时间也不适合单独计算精确耗时。

**可以这样答：**

> 单进程耗时用不受系统时间回拨影响的单调时钟计算，跨服务关联保留墙上时间和 Trace 因果关系，并通过时间同步控制而不是假设所有机器绝对同钟。例如，模拟 NTP 校时和容器迁移，确认本地耗时不为负、跨服务瀑布图仍能依靠父子 Span 解释。需要注意的是，单调时钟值不能跨进程直接相减，墙上时间也不适合单独计算精确耗时。

## 常见追问

1. **“延迟计时与时钟偏差”最需要讲清的核心内容是什么？** 单进程耗时用不受系统时间回拨影响的单调时钟计算，跨服务关联保留墙上时间和 Trace 因果关系，并通过时间同步控制而不是假设所有机器绝对同钟
2. **哪项具体检查可以支撑你对“延迟计时与时钟偏差”的判断？** 模拟 NTP 校时和容器迁移，确认本地耗时不为负、跨服务瀑布图仍能依靠父子 Span 解释
3. **“延迟计时与时钟偏差”最容易被忽略的前提是什么？** 单调时钟值不能跨进程直接相减，墙上时间也不适合单独计算精确耗时
