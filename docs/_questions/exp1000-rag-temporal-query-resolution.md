## 面试时怎么答

回答时间意图解析、文档有效区间过滤和事件时间与入库时间区分。

**可以这样答：**

> 先把“当时”解析为明确时间范围，无法从对话确定时向用户追问。文档索引要保存内容生效时间和失效时间，不能只使用抓取或入库时间。查询按目标时间过滤有效版本，再按相关性排序，并把采用的时间假设展示给用户。若知识库只有当前版本，应明确无法可靠还原历史，而不是用今天的规则回答过去。

## 常见追问

1. **发布时间等于生效时间吗？** 不一定，政策可能延后生效或追溯生效，两者要分字段保存。
2. **用户问“最近”怎么处理？** 根据产品定义转换为时间窗口，并在答案中说明具体起止日期。
