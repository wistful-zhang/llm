## 面试时怎么答

把搜索、邮件和网页结果视为不可信数据，说明结构化提取和权限不随内容改变。

**可以这样答：**

> 工具返回的网页、邮件或文档最终也进入模型上下文，模型可能把其中命令误当下一步行动。每个字段要标注来源与信任级别，正文只用于提取任务所需事实，不能修改系统策略。高风险参数必须来自用户确认或可信系统字段，不能直接从不可信正文复制。后续工具调用继续经过独立授权和校验，即使内容声称已获得管理员批准也无效。

## 常见追问

1. **先让另一个模型清洗工具结果可以吗？** 可降低部分风险，但清洗模型也会被攻击，不能替代权限隔离。
2. **结构化 API 返回就一定可信？** 不一定，字符串字段仍可能包含恶意内容，且 API 本身也可能被污染。
