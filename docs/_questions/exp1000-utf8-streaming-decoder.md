## 面试时怎么答

口述“流式字符解码”时，先回答它是什么或怎么做，再用例子落地，最后交代适用限制：Unicode 字符边界仍不同于用户可见字素，光标和计数还需更高层处理。

**可以这样答：**

> 网络块或 token 字节可能在多字节字符中间切开，增量解码器应保留未完成字节直到后续到达。一个直接的检查办法是：把中文、Emoji 和组合字符随机切块，验证重组后完全一致。这个结论的边界是，Unicode 字符边界仍不同于用户可见字素，光标和计数还需更高层处理。

## 常见追问

1. **请用自己的话说明“流式字符解码”的核心做法。** 网络块或 token 字节可能在多字节字符中间切开，增量解码器应保留未完成字节直到后续到达
2. **你准备怎样举例证明自己理解“流式字符解码”？** 把中文、Emoji 和组合字符随机切块，验证重组后完全一致
3. **使用“流式字符解码”前还要确认什么？** Unicode 字符边界仍不同于用户可见字素，光标和计数还需更高层处理
