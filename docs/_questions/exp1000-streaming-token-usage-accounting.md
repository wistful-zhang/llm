## 面试时怎么答

**可以这样答：**

> 核心判断是，区分预留额度、本地估算和供应商最终账单；正常结束、客户端取消与上游断流都记录独立终态和可追溯 request ID。实际验证可采用这个办法：模拟完整结束、半途取消和缺少 usage 终帧，核对额度释放与账单对账。此外，本地 tokenizer 估算不一定等于供应商计费，失败请求是否收费也取决于明确契约。
