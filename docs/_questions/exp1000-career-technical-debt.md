## 面试时怎么答

这类题不要套用别人的经历。围绕“技术债管理”讲清背景、目标、本人动作与真实证据，最后说明没有负责人和期限的 TODO 不是计划，不能把长期缺陷都归为技术债。

**可以这样答：**

> 明确临时方案、风险、Owner、触发条件和截止时间，把偿还项放入可见计划并用指标证明完成。证据部分只写本人真实材料：替换为本人真实临时决策、故障风险和后续重构。还要明确，没有负责人和期限的 TODO 不是计划，不能把长期缺陷都归为技术债。

## 常见追问

1. **回答“技术债管理”时应覆盖哪些本人事实？** 明确临时方案、风险、Owner、触发条件和截止时间，把偿还项放入可见计划并用指标证明完成
2. **可以用什么真实证据支撑“技术债管理”？** 替换为本人真实临时决策、故障风险和后续重构
3. **复盘“技术债管理”时必须主动承认哪项边界？** 没有负责人和期限的 TODO 不是计划，不能把长期缺陷都归为技术债
