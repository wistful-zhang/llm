# 隐私说明

这个项目是静态网站，没有本站账号、服务器数据库或行为统计。题库阅读不要求登录；面试记录和模拟练习进度默认保存在浏览器本地，不会自动提交到 GitHub。

## 浏览器里保存什么

- **面试记录**：公司匿名代号、岗位、轮次、结果、下一步和复盘，保存在 `localStorage`。
- **模拟练习**：未完成场次和最近一次复盘结果，保存在 `localStorage`。
- **保存多久**：浏览器会一直保留，直到你主动清除、浏览器清理站点数据，或存储空间发生变化。
- **怎样删除**：面试记录页的“彻底删除本机记录”会同时删除当前数据和恢复副本；浏览器设置也可以删除整个站点的数据。

“仅当前浏览器”不是账号鉴权或加密。同一浏览器资料中的其他使用者可能看到记录；同一个 `username.github.io` 域名下的其他项目也共享浏览器 Origin，理论上可以读取相同 Origin 的本地存储。请使用公司匿名代号，不要保存姓名、联系方式、会议链接、精确薪资、公司机密或受 NDA 约束的信息。更高隔离要求应使用独立域名、独立浏览器资料或专门的加密记录工具。

## 备份文件

导出的 JSON 和 CSV 是未加密明文文件。JSON 用于完整恢复，CSV 用于表格查看；下载后由你负责存放、传输和删除。换设备、仓库改名、转移 Owner 或切换网站地址前，请先导出 JSON，再到目标站点恢复。

## 页面会请求哪些外部服务

- 题目公式按需从 jsDelivr 加载 MathJax 文件和字体。
- 创建题库按钮会请求 GitHub 的公开仓库 API，用于判断应复制当前模板还是基础模板；请求不包含本站面试记录。
- “配置自检”会请求 GitHub 的公开仓库与 Actions API，读取仓库可见性、默认分支、Pages 状态和公开工作流运行结果；无需也不会收集 Token，不会读取本站面试记录。
- 点击“GitHub 仓库”“投稿与纠错”或 Pages CMS 后，会离开本站并受对应服务的隐私政策约束。

本站没有配置广告、第三方统计或站内 Cookie。GitHub Pages、浏览器和你主动打开的外部服务仍可能按各自规则处理网络日志。

## Pages CMS 与 GitHub

Pages CMS 是可选的第三方编辑界面。授权后，它可以读取和修改你选中的仓库，并可能缓存仓库配置、内容和权限信息；Private 仓库内容也会经过获授权的服务。建议只授权题库仓库，并定期在 GitHub 的 **Settings → Applications → Installed GitHub Apps** 中复查或撤销权限。不使用 Pages CMS 时，可以直接在 GitHub 网页编辑器中维护文件。

任何进入 Public 仓库、Issue、Pull Request 或 GitHub Pages 的内容都应视为公开。网页上的“未发布”开关不能隐藏 Public 仓库中的源文件或 Git 历史。
