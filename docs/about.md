# 关于这份笔记

这里记录我准备 LLM 面试时遇到的问题，也希望它能帮助更多正在学习大模型的朋友。

## 整理原则

- **先给结论**：面试时能在 30 秒内回答核心问题。
- **再讲原理**：说明为什么，而不只是背诵定义。
- **联系实践**：补充真实工程中的选择和取舍。
- **准备追问**：把面试官可能继续深挖的方向列出来。

## 如何使用

先尝试自己回答题目，再查看核心回答；接着用“展开说明”检查知识盲点，最后逐个回答“常见追问”。
