# 关于这份笔记

这是一个可复用的大模型面试题库与网站模板。你可以浏览当前题库，也可以使用同一套项目建立属于自己的公开或私人题库。

## 整理原则

- **先给结论**：面试时能在 30 秒内回答核心问题。
- **再讲原理**：说明为什么，而不只是背诵定义。
- **联系实践**：补充真实工程中的选择和取舍。
- **准备追问**：把面试官可能继续深挖的方向列出来。

## 如何使用

先尝试自己回答题目，再查看核心回答；接着用“展开说明”检查知识盲点，最后逐个回答“常见追问”。

如果想建立自己的题库，请查看[使用指南]({{ '/start/' | relative_url }})。整个项目不依赖固定用户名或仓库名。
