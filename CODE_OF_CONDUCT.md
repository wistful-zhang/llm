# 社区行为准则

## 我们的承诺

我们致力于为所有参与者提供友善、开放、尊重且无骚扰的协作环境，不因经验水平、教育背景、身份、外貌、能力或观点不同而区别对待。

## 鼓励的行为

- 以事实和问题为中心，给出清晰、建设性的反馈
- 尊重不同学习路径、技术选择和表达方式
- 主动说明不确定性，接受纠正并及时修正错误
- 保护贡献者隐私，不传播非公开面试内容或公司机密
- 在分歧中保持克制，优先寻找可验证的共同结论

## 不可接受的行为

- 人身攻击、侮辱、歧视、骚扰或威胁
- 未经许可公开他人的个人信息或私密沟通
- 发布密码、Token、公司机密或受保密协议约束的材料
- 垃圾广告、恶意灌水或持续扰乱项目协作
- 其他在专业协作环境中明显不恰当的行为

## 执行

维护者有责任解释和执行本准则，可以编辑、隐藏或删除不当内容，并根据情况警告或限制参与者。报告行为问题时，请使用维护者 GitHub 个人资料中提供的私密联系方式，不要在公开 Issue 中披露受影响者的隐私。

所有报告都应得到及时、客观和保密的处理。维护者不得报复善意报告者。
